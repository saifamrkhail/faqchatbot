# Planning Artifact Index

This directory contains the normalized planning set for the project.

## Artifact Set

1. `PROJECT-DEFINITION.md`
   - Stable project scope, quick overview, constraints, and success criteria.
2. `MODULES.md`
   - Module map, responsibilities, boundaries, and implementation order.
3. `IMPLEMENTATION-PLAN.md`
   - Phase-based roadmap with deliverables, dependencies, and exit criteria.
4. `modules/`
   - One brief per implementation module.

## Recommended Reading Order

1. `PROJECT-DEFINITION.md`
2. `MODULES.md`
3. `IMPLEMENTATION-PLAN.md`
4. `modules/`

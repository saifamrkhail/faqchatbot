# Module 08 - Terminal UI

## Purpose

Provide a usable Textual interface over the chat application service.

## Scope

- chat layout
- input field and submit flow
- response rendering
- loading and status feedback
- controlled error display

## Expected Deliverables

- `app/ui/app.py`
- Textual widgets or screens as needed
- integration with the chat application service

## Dependencies

- Module 07

## Done When

- a user can ask a question in the terminal and see the answer
- long responses remain readable
- UI behavior stays stable during backend failures
